# Zepto Sales Analysis Dashboard

## Project Overview
This Power BI dashboard provides a comprehensive analysis of sales data for Zepto, focusing on revenue trends, regional performance, and customer preferences. It is designed to help stakeholders identify high-performing cities and popular cuisine categories to optimize delivery operations and marketing strategies.

## Key Features
*   **Performance Metrics:** Real-time tracking of Total Sales (987M), Sales Quantity (2M), and Customer Ratings (148K).
*   **Geographic Analysis:** A ranked bar chart showing sales performance across major Indian cities.
*   **Dynamic Threshold Filtering:** A custom button slicer that allows users to filter cities based on number of cities (0, 5, 10, 20, 50, 100) for targeted analysis.
*   **Cuisine Insights:** Breakdown of sales by food category to identify market demand.
*   **Historical Trends:** An area chart visualizing sales fluctuations from 2017 to 2020.

## Tech Stack & Skills
*   **Tool:** Microsoft Power BI
*   **Data Transformation:** Power Query
*   **Calculations:** DAX (Data Analysis Expressions) for complex measures and dynamic filtering.
*   **Soft Skills:** Data Visualization, Analytical Thinking, Business Intelligence.

## How to Use
1.  **Overview:** Check the top KPI cards for a quick snapshot of the business.
2.  **Filter Cities:** Use the button slicer at the top right to limit the city bar chart to specific sales volumes.
3.  **Trend Analysis:** Observe the 'Year Sales' area chart to understand year-over-year growth and declines.
